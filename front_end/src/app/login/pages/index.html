Perguntas frequentes

Tirando dúvidas comuns

Como o cliente acessa a O.S. dele?

A oficina gera um código de acesso de 8 caracteres a partir do botão "Código do cliente" na tela da OS. O cliente entra em Acessar minha O.S. informando o código + CPF/CNPJ. O código tem validade (padrão 7 dias), limite de tentativas e pode ser revogado a qualquer momento.

Preciso instalar algum aplicativo?

Não. O Pit Stop é 100% web. Funciona no navegador do computador, tablet ou celular — e respeita preferências de acessibilidade do sistema (tema escuro, redução de movimento, etc.).

É seguro? Quem vê o quê?

Há 5 papéis pré-definidos (admin, gerente, atendente, mecânico, visualizador) e cada endpoint da API valida o papel. Operações sensíveis ficam registradas em log de auditoria, e o cliente final só enxerga a OS vinculada ao código informado.

Como cadastrar minha oficina?

Clique em Cadastrar minha oficina. O cadastro leva menos de 2 minutos: nome, CNPJ, endereço, dias de funcionamento e os dados do administrador. O primeiro usuário criado vira automaticamente o admin da sua oficina.

Quem pode acessar o painel administrativo SaaS?

Apenas usuários com flag staff ou superuser (criados via manage.py createsuperuser ou pelo próprio painel SaaS) podem ver oficinas e usuários globais. Os funcionários comuns nunca enxergam dados de outras oficinas.
